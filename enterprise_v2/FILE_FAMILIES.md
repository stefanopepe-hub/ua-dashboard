# File Families

## Famiglie supportate
- saving_orders
- detailed_orders
- resources_team
- cycle_times
- non_conformities
- suppliers_master

## Regola
Il sistema deve riconoscere automaticamente la famiglia file tramite:
- nome foglio
- nomi colonna
- sinonimi
- pattern dei valori
- contesto del file

## Obiettivo
Non dipendere da un template rigido.
