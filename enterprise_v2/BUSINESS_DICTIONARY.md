# Business Dictionary

## Document Types - Ricerca
- ORN = Ordine Ricerca
- ORD = Ordine Diretto Ricerca
- OPR = Ordine Previsionale Ricerca
- PS = Procedura Straordinaria

## Document Types - Struttura
- OS = Ordine Struttura
- OSP = Ordine Previsionale Struttura
- OSD = Ordine Diretto Struttura

## Regola
Queste etichette devono essere usate in:
- import
- dimensioni canoniche
- filtri
- tabelle
- grafici
- report
