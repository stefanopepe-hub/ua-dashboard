# Multi Sheet Plan

## Situazione attuale
Il reader legge solo il primo foglio utile del file Excel.

## Prossimo obiettivo
Aggiungere ispezione multi-sheet per:
- elencare i fogli presenti
- leggere le colonne di ciascun foglio
- scegliere il foglio più rilevante
- supportare file complessi esportati da ERP o rilavorati dagli utenti

## Criteri futuri di scelta foglio
- numero colonne mappabili
- presenza di sinonimi chiave
- presenza di campi obbligatori
- confidence score più alto
