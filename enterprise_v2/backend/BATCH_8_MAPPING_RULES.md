# Batch 8 Mapping Rules

## Regole aggiunte
- trattare . _ / - come separatori equivalenti
- usare fallback semantico se compare 'negoziato'
- usare fallback semantico se compare 'iniziale'
- riconoscere 'data doc.' come document_date
- riconoscere protocolli anche in forma abbreviata

## Nota
Queste regole servono a supportare file Excel reali con intestazioni variabili.
